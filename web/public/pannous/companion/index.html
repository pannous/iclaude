The Companion

A web UI for Claude Code & Codex.

Run multiple AI coding sessions in your browser with full streaming, tool call visibility, and permission control. The Companion reverse-engineers the undocumented WebSocket protocol of Claude Code CLI to provide a rich, multi-session interface.

bunx iclaude

Features

  • Multi-session management — run and switch between sessions simultaneously
  • Real-time streaming via WebSocket bridge
  • Permission control — approve or deny tool calls from the browser
  • Inline HTML fragments with bidirectional state
  • Session persistence across server restarts
  • Protocol recordings for debugging and replay
  • Tunnel support via Tailscale Funnel
  • Token-based authentication for secure remote access
  • Works with both Claude Code and Codex

How It Works

The server (Hono + Bun) sits between your browser and the Claude Code CLI. It spawns CLI processes, bridges their NDJSON WebSocket protocol to typed JSON for the frontend, and manages session lifecycle — all on a single port. No external databases or services required.
